Use Case: Connect 4 Win

Use Case

  • Win a game of Connect 4

    • Primary Actor

  • Player (visitor of game site)

    • Scope

  • CMPS 262 Connect 4 Game

    • Level

  • Goal is to stop the game and announce a winner when a win occurs

    • Brief

  • The player drops four of the same color disk in a row vertically, horizontally, or diagonally in order to win the game. Winning ends the game and provides an option to restart.

    • Stakeholders

  • Player 1
  • Player 2

    • Postconditions

    Minimal Guarantees

  • Game Ends

    • Success Guarantees

  • Winner is announced
  • Button is displayed to start a new game

    • Preconditions

  • Player completes game.
  • Four pieces of same players disc line up in a row.

    • Basic Flow

    1. The player drops the fourth disc of the same color in a row horizontally, vertically, or diagonally.
    2. The game stops allowing pieces to be dropped.
    3. The site announces the winning player.
    4. A button is displated indicating to start a new game.
